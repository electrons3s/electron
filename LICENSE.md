## 🛠️ Troubleshooting Common Issues

- ❌ **Executor won't attach to Roblox**
  The game window must be fully loaded before you press `Attach`. Wait until you are standing in the experience, then try again. If the problem persists, increase `injection_delay_ms` to `500` in your `config.json`.
  **Solution:** Restart both Roblox and **Electron Executor**, ensure you are running the latest version, and attach only after the game world is visible.

- ❌ **Antivirus deletes the installer**
  Injection-based tools are frequently flagged as false positives by antivirus engines. This does not mean the software is harmful — it reflects how the detection heuristics work.
  **Solution:** Add the **Electron Executor** install directory to your antivirus exclusion list, then re-download the file from the **Electron Executor Github** page.

- ❌ **iOS profile expires or becomes untrusted**
  Apple revokes sideloaded profiles periodically, especially after system updates, which breaks **Electron Executor Ios Direct Install**.
  **Solution:** Delete the old profile in `Settings > General > VPN & Device Management`, re-download the profile from the official page, and re-trust it.

- ❌ **Scripts execute but nothing happens in-game**
  The script may target a different game or an outdated API. Alternatively, **Electron Executor** might not have attached correctly.
  **Solution:** Confirm the script is designed for the game you are in, re-attach **Electron Executor**, and check the Debug Console for error messages.

- ❌ **Android app crashes on launch**
  Older Android versions or heavy manufacturer skins can interfere with the **Electron Executor Mobile Apk** runtime.
  **Solution:** Make sure you are running Android 9.0 or higher, clear the **Electron Executor** app cache, and reinstall the latest `.apk` from the **Electron Executor Official** source.
